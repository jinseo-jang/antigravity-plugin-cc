# Security Policy

## Supported versions

Only the latest `0.1.x` release (the `main` branch) is supported.

## Reporting a vulnerability

**Do not** report security issues through public GitHub issues, discussions, or pull requests.

Report privately through GitHub Private Vulnerability Reporting:

👉 https://github.com/jinseo-jang/antigravity-plugin-cc/security/advisories/new

> That link only accepts reports once the maintainer has enabled **Settings → Code security → Private vulnerability reporting** on the repo. If the link doesn't work (or you can't use the GitHub UI), open a public issue titled `security: please enable private reporting` **without any vulnerability details**, and the maintainer will turn it on.

Please include:

- what the issue is and its impact,
- steps to reproduce or a minimal proof of concept,
- the affected version or commit SHA.

This is a spare-time project with no bug-bounty program — please allow reasonable time for a fix before any public disclosure.
